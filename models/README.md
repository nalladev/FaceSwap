# Model Files Directory

This directory should contain the required model files for face detection and recognition:

- `shape_predictor_68_face_landmarks.dat` - Dlib facial landmark predictor
- Other model files as needed

## Download Instructions

Download the required model files from the official sources:

1. Dlib shape predictor: http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

Extract and place the files in this directory for the tests to pass.
